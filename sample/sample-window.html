Dialog

This sample demonstrate how to use lgl_dialog to replace browser alert/confirm/input

Usage

<script type="text/javascript">
    LGL.UI.alert("Hello world");
    LGL.UI.confirm("Do like this library?", {
        onYes: function() {
            LGL.UI.alert("Oh really? Thank you!");
        },
        onNo: function() {
            LGL.UI.alert("oops, sorry about that...");
        }
    });
    LGL.UI.input("What's your name?", {
        onOkay: function() {
            LGL.UI.alert("Good on you, " + this.value + "!");
        }
    });
</script>

Tool tip

This sample demonstrate how to use lgl_tip to replace browser tooltip with your customized tip

Usage

There is nothing special need to do in your html file, just add "title='your tip'" as normal to your element

<a href="http://www.google.com" title="aha, where do you suppose to go?">Yahoo</div>

Make sure you include lgl_tip.js in your document

<script type="text/javascript" src="/path-to-js/lgl_tip.js"></script>
